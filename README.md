### Instalacióm

Primero, se clona el siguiente repositorio:

```bash
git clone https://github.com/GuillermoCuesta/CINE-PACHO-Front.git
```

Segundo, descargar los paquetes: 

```
npm install
```
Tercero, correr el front en local:

```
npm run dev
```
